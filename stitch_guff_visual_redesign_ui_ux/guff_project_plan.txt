# GUFF UI Redesign Project

## Overview
GUFF is a complete E2EE messaging + social platform redesign.

## Core Screens to Build
1. **Auth Page (`/`)**: Split layout, branding panel (dark indigo gradient) vs form panel.
2. **App Layout**: Collapsible sidebar with smooth transitions, logo top, user bottom.
3. **Feed Page (`/feed`)**: Stories tray (gradient rings), Create Post card (expanding), Post Cards (masonry media).
4. **Messages Layout (`/messages`)**: Split pane with conversation list (online indicators, unread counts).
5. **Chat Thread (`/messages/[id]`)**: Header with call icons, sent/received bubbles with read receipts, typing indicator, rounded composer.
6. **Explore Page (`/explore`)**: Prominent search, trending pills, user grid.
7. **Profile Page (`/profile/[id]`)**: Cover photo, large avatar, stats, masonry post grid.
8. **Video Call Overlay**: Dark fullscreen, floating speaker, grid view, connection indicators.
9. **Settings Slide-over**: Profile editing, privacy toggles, push notifications.

## Key Design Principles
- **Motion**: Framer Motion transitions, spring animations for hearts/buttons, pulsing rings for calls.
- **Visuals**: Squircle avatars, indigo gradients, frosted glass blur, Inter typography.
- **UX**: Clear empty states, skeleton loaders, floating labels.
